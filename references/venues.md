
# 各会议/期刊格式要求与审稿偏好

## 计算机视觉顶会

### CVPR (IEEE Conference on Computer Vision and Pattern Recognition)

- **投稿截稿**：通常11月
- **页数限制**：正文8页 + 参考文献（无上限）（2025-2026确认不变）
- **格式**：IEEE双栏，LaTeX模板 cvpr.sty
- **审稿周期**：约3个月
- **审稿倾向**：
  - 实验必须充分，至少3个benchmark
  - 定性可视化质量很重要
  - 附带补充材料（supplementary）常见
  - 喜欢有实际应用场景的motivation
- **页码格式**：2024年起使用e-ISBN页码（如10349-10358）
- **CVPR 2025**：投稿13008篇，录用2878篇（录用率22.1%）

### ICCV (IEEE/CVF International Conference on Computer Vision)

- **投稿截稿**：通常3月
- **页数限制**：正文8页 + 参考文献
- **格式**：IEEE双栏
- **审稿倾向**：
  - 技术深度要求略高于CVPR
  - 理论分析是加分项
  - 更关注方法的fundamental contribution

### ECCV (European Conference on Computer Vision)

- **投稿截稿**：通常3月（与ICCV交替）
- **页数限制**：正文14页（LNCS格式，单栏） + 参考文献
- **格式**：Springer LNCS，单栏
- **审稿倾向**：
  - 页数宽裕，可以展开更详细的方法描述
  - 允许更长的相关工作综述

## 计算机图形学顶会

### SIGGRAPH (ACM SIGGRAPH)

- **投稿截稿**：通常1月
- **页数限制**：Journal Track: 约8页；Conference Track: 会议论文格式
- **格式**：ACM TOG格式
- **审稿倾向**：
  - 技术深度和数学严谨性是首要标准
  - 审稿人有图形学背景，对渲染方程、几何处理要求专业
  - 视觉效果和demo video很重要
  - 偏好有理论保证的方法（收敛性、复杂度分析）

### Eurographics (EG)

- **页数限制**：通常10-12页
- **格式**：Computer Graphics Forum (CGF)
- **审稿倾向**：
  - 欧洲审稿人偏重理论
  - 对方法的普适性和鲁棒性关注较高

### Pacific Graphics (PG)

- **页数限制**：8页
- **审稿倾向**：
  - 接受范围比SIGGRAPH/EG更广
  - 适合有潜力但尚不成熟的工作

## 机器学习顶会

### NeurIPS (Conference on Neural Information Processing Systems)

- **投稿截稿**：通常5月
- **页数限制**：正文9页 + 附录
- **格式**：NeurIPS-relevant format
- **审稿倾向**：
  - 理论贡献权重很高
  - 偏好方法有理论保证
  - 实验是加分项但非必需
  - 口头报告常见

### AAAI (AAAI Conference on Artificial Intelligence)

- **投稿截稿**：通常8月
- **页数限制**：正文7页 + 附录
- **审稿倾向**：
  - 接受范围广，适合有应用价值的工作
  - 偏好清晰的技术贡献
  - 不要求深理论但需要实验验证

## 核心期刊

### IEEE TVCG (Transactions on Visualization and Computer Graphics)

- **影响因子**：~5.2
- **页数**：无严格限制，通常12-18页
- **审稿周期**：6-12个月
- **特点**：覆盖可视化和图形学，JCGRT格式

### CGF (Computer Graphics Forum)

- **影响因子**：~2.5
- **页数**：通常10-15页
- **特点**：EG关联期刊，Wiley出版

### ACM TOG (ACM Transactions on Graphics)

- **影响因子**：~6.7（图形学最高）
- **特点**：SIGGRAPH/EG论文的期刊扩展版

### IEEE TPAMI

- **影响因子**：~24
- **页数**：14页
- **特点**：偏重理论和方法论，审稿极严
- **适合**：有理论贡献的扩展版工作（如Ref-NeRF）

## 博士论文格式要点

- 封面、摘要（中英文）、目录、符号表
- 每章末尾有"本章小结"
- 创新点声明在引言中明确列出
- 致谢单独成页
- 参考文献通常100+篇
- 附录可放补充实验、证明推导
